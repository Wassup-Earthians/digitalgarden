# 13/02/2026 (Friday)

Applying the approach carried out in Multisim back to Tinkercad should have worked but it was not working as intended. After tinkering the circuit several times, we decided to use another software for our task. We searched for contenders that cam simulate basic electronics equipment such as oscilloscope, function generator, while also supporting integration of microprocessor equipments like Arduino, ESP32 or STM32, etc.
Finally we chose Proteus for our job.
